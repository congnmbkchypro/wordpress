A fashionate theme called wordpress fashion theme and blog.

== Description ==

Selecting a well-structured WordPress theme helps your fashion blog to rank well on major search engines. Fashionate is simple and easy to use, clean WordPress theme perfectly fit on your blog website.

* Mobile-first, Responsive Layout with real device
* Custom Colors
* Custom Header
* Menu Description
* Post Formats


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
5. In your admin panel, go to Plugins > and click the Add New button for install " Contact Form 7 ".
6. Go to https://www.themexpert.com/docs/wordpress-themes/fashionate for a guide on how to customize this theme.
